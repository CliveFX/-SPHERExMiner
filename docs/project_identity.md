# Project Identity

Working project name: **LuxQuarry**.

Rationale:

- `Lux` keeps the project tied to light, photons, and spectral measurements.
- `Quarry` captures the mining/search aspect without locking the project to one
  telescope or data release.
- The name can cover the full stack: SPHEREx mining, injection/recovery,
  candidate scoring, dashboards, and later distributed survey runs.

Suggested naming pattern:

- Project: `LuxQuarry`
- Main miner: `luxquarry-miner`
- Candidate scorer: `luxquarry-scorer`
- Dashboards/viewers: `luxquarry-viewer`
- Future survey runner: `luxquarry-survey`

Current repo/package names still use the original SPHEREx Laser Miner naming.
Renaming should wait until the science/performance interfaces stabilize.
